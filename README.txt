data:
	Contains all data, both raw and edited. Has several subfolders.
	The immediatly visible files are tidied tables for the different types of aircraft as CSV-files as well as CSV-files with the operators that recieved the aircrafts from new (in most cases meaning the operators who ordered the aircraft).

	raw:
		All raw data including original zip-file and unwrapped zip-file.

	ready-for-use:
		Slightly poorly named; it is the folder with CSV-files that are ready to be tidied.

	sources:
		Where the data for the project has been found.

script:
	Contains small scripts in python that have been used for processing the data.

output:
	Contains the visualisations/figures.
